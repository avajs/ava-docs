# S'assurer que `t.end()` est seulement appelé dans `test.cb()`

AVA échouera si `t.end()` est appelé dans une fonction de test qui n'est pas `cb`.


## Échoue

```js
import test from 'ava';

test('quelques tests', t => {
	t.pass();
	t.end();
});
```


## Passe

```js
import test from 'ava';

test('quelques tests', t => {
	t.pass();
});

test.cb('quelques tests', t => {
	t.pass();
	t.end();
});
```
