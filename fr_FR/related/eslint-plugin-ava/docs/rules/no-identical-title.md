# S'assurer que chaque test a un titre différent

Interdit les tests avec des titres identiques car il est difficile de les différencier.


## Échoue

```js
import test from 'ava';

test('foo', t => {
	t.pass();
});

test('foo', t => {
	t.pass();
});
```


## Passe

```js
import test from 'ava';

test(t => {
	t.pass();
});

test('foo', t => {
	t.pass();
});

test('bar', t => {
	t.pass();
});
```
